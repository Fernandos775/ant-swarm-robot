# MindSwarm - Рой роботов-муравьёв на ESP32-CAM

Проект по созданию swarm-системы мини-роботов. Первый прототип — наземный муравей (GroundBot) с управлением через Serial Monitor.

## Что уже работает
- Шасси 4WD с 4 моторами
- Управление моторами через ESP32-CAM
- Подключение к Wi-Fi
- Базовое движение (вперёд, назад, повороты, стоп)

## Технологии
- **Микроконтроллер**: ESP32-CAM (Ai-Thinker)
- **Язык**: C++ (Arduino IDE)
- **Драйвер моторов**: L298N / MX1508
- **Связь**: Wi-Fi

## Как запустить
1. Открой `code/ant_v1_serial_control.ino` в Arduino IDE
2. Выбери плату **AI Thinker ESP32-CAM**
3. Загрузи код
4. Открой Serial Monitor (115200 baud)
5. Используй команды: `f` `b` `l` `r` `s`
